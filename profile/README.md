# 🚀 Roadmap: Desenvolvimento Web

Bem-vindo ao repositório central de estudos e projetos da organização.

## 🎯 Objetivo
Dominar a stack full-stack moderna, partindo da base do CS50x de Harvard até a implementação de sistemas.

---

## 🗺️ O Roadmap

### 🧱 Fase 1: Fundamentos (A Base do CS50x)
*O foco é o entendimento profundo de como o software interage com o hardware.*
- [ ] **C:** Sintaxe, tipos e fluxos de controle.
- [ ] **Arrays:** Estruturação de dados contíguos.
- [ ] **Algoritmos:** Ordenação, busca e eficiência (Big O).
- [ ] **Memória:** Gestão manual e ponteiros.
- [ ] **Estrutura de Dados:** Listas, árvores e tabelas hash.

### 🗄️ Fase 2: Lógica de Dados e Backend
*Transição para alto nível e persistência de dados.*
- [ ] **Python:** Produtividade e lógica avançada.
- [ ] **Modelagem de Banco de Dados:** Entidades e relacionamentos.
- [ ] **SQL:** Linguagem de consulta estruturada.
- [ ] **Postgres:** Administração e otimização de banco relacional.

### 🌐 Fase 3: Web Core & Workflow
*Ferramentas essenciais para o dia a dia do desenvolvedor.*
- [ ] **Git e GitHub:** Controle de versão e colaboração.
- [ ] **HTML, CSS, Javascript:** A base da experiência do usuário.

### ⚛️ Fase 4: Ecossistema Frontend
*Criação de interfaces modernas e performáticas.*
- [ ] **ViteJS:** Ferramental de build ultra-rápido.
- [ ] **React:** Desenvolvimento baseado em componentes.
- [ ] **NextJS:** SSR, roteamento e frameworks full-stack.

### ⚙️ Fase 5: APIs de Alta Performance
*Construção de serviços robustos.*
- [ ] **FastAPI:** APIs assíncronas e tipagem forte com Python.
- [ ] **NoSQL:**: Bancos não relacionais para alta escalabilidade e esquemas flexíveis.
- [ ] **Meilisearch**: Motor de busca ultrarrápido e tolerante a erros de digitação.

### 🤝 Fase 6: Prática Colaborativa
*Aplicação de conhecimentos em ambientes reais.*
- [ ] **SEMANA DA ATIVIDADE PRÁTICA EM DUPLA:** Desenvolvimento em par e code review.

### 🏗️ Fase 7: Arquitetura e Engenharia de Harness
*O estado da arte na produção de software.*
- [ ] **Docker:** Containerização e reprodutibilidade.
- [ ] **Cache:** Estratégias de latência e performance.
- [ ] **System Design:** Arquitetura de sistemas distribuídos.
- [ ] **Engenharia de Harness:** Implementação de fluxos de trabalho onde agentes de IA (LLMs) geram código em escala dentro de scaffolds e limites pré-definidos.

### 🏆 Fase 8: Consolidação (The Capstone)
*Entrega de valor de ponta a ponta.*
- [ ] **ATIVIDADE FINAL:** Projeto Full-Stack!
